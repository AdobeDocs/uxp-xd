## setInterval(callback, delta)
Creates a request to call `callback` repeatedly every `delta` milliseconds.
Cancelable using the return ID.

**Returns**: `\*` - Interval ID

**Category**: browser, timers

| Param | Type |
| --- | --- |
| callback | `function` |
| delta | `number` |
