## CSSMediaRule

**Extends**: [`CSSGroupingRule`](#cssgroupingrule)

**Category**: browser, cssom

### cssMediaRule.type
Gets the type of this rule

**Read only**



### cssMediaRule.media : [`CSSMediaList`](#cssmedialist)
Returns the list of media rules

**Read only**



### cssMediaRule.mediaText : `DOMString`

**Read only**
