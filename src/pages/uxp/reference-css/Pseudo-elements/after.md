# ::after

**Since:** UXP v3.0

Creates a pseudo-element after the matched element. 

**See:** https://developer.mozilla.org/en-US/docs/Web/CSS/::after

**Example:**

```css
.withCommas > *::after {
    content: ",";
}
```
