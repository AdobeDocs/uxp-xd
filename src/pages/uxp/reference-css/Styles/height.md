# height

**Since:** UXP v2.0

Specifies the height for an element.

**See**: [https://developer.mozilla.org/en/docs/Web/CSS/height](https://developer.mozilla.org/en/docs/Web/CSS/height)

**Example**:

```css
.someElement {
    height: 100vh;
}
```
