# overflow-y

**Since:** UXP v3.0

Shorthand which determines what is done with content that overflows the element's bounds.

Supports `visible`, `hidden`, `auto`, and `scroll`.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/overflow-y
