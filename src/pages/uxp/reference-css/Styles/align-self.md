# align-self

**Since:** UXP v3.0

Supported values are `flex-start`, `flex-end`, `center`, `stretch`.


**See**: https://developer.mozilla.org/en/docs/Web/CSS/align-self

**Example**

```css
.someElement {
    align-self: flex-start;
}
```
