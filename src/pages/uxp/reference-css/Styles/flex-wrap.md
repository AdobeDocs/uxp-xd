# flex-wrap

**Since:** UXP v3.0

Controls if wrapping is allowed within a flexible container. Supports `nowrap` (default) and `wrap`.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/flex-wrap

**Example**

```css
.someElement {
    flex-wrap: wrap;
}
```
