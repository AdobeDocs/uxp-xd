# top

**Since:**  UXP v2.0

Sets the top position for an element that is relatively or absolutely positioned.

**See**: [https://developer.mozilla.org/en/docs/Web/CSS/top](https://developer.mozilla.org/en/docs/Web/CSS/top)

**Example**:

```css
.someElement {
    top: 0;
    position: absolute;
}
```
