# margin-bottom

**Since:**  UXP v2.0

Specifies the bottom margin for an element.

**See**: https://developer.mozilla.org/en/docs/Web/CSS/margin-bottom

**Example**:

```css
.someElement {
    margin-bottom: 1em;
}
```

## Quirks and Exceptions

* Margins do not collapse in UXP.
