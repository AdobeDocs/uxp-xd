# :last-child

**Since:** UXP v3.0

Matches only the last child.

**See:** [https://developer.mozilla.org/en-US/docs/Web/CSS/:last-child](https://developer.mozilla.org/en-US/docs/Web/CSS/:last-child)

**Example:**

```css
sp-button:last-child {
    margin-right: 0;
}
```
